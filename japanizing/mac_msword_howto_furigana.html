Mac: How to add Furigana (MSWord)
ふりがなのつけ方


Office → MSWord → additional tools → Microsoft language register → Microsoft language register

  

“select the language to enable for Microsoft Office” → Japanese → OK

  

Open MSWord → highlight a kanji compound → format → Asian layout → phonetic guide

or

Open MSWord → highlight a kanji compound →"Home" menu bar --> click
